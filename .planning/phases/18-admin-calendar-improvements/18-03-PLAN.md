<objective>
Add conditional address field visibility to the Create Booking modal based on `serviceDeliveryModel` from company settings.

Purpose: Cleaning services come to the customer's location (`at-customer`) — the address is essential. If the business model changes to `customer-comes-in` (e.g., a salon), the address field is irrelevant and creates noise. This change reads the live company setting and conditionally shows/hides the field.

Output: The `customerAddress` schema becomes optional, `useCompanySettings()` is imported and used to derive `showAddressField`, the address `FormField` renders conditionally, and a `useEffect` clears the value when the field is hidden.
</objective>

const { settings: companySettings } = useCompanySettings();
