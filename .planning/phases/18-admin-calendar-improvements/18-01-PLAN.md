<objective>
Fix the visual misalignment between react-big-calendar time-gutter labels and their corresponding horizontal grid lines in the admin calendar.

Purpose: React-big-calendar renders `.rbc-label` elements at the top of each timeslot group rather than centred on the grid line, causing labels like "9:00 AM" to appear above the line they label. A CSS transform corrects this.

Output: A 2-line CSS addition to `client/src/index.css` that moves `.rbc-label` text into alignment with the grid line.
</objective>
