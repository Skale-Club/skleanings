express-rate-limit v8 API:
```typescript
import rateLimit from "express-rate-limit";

const limiter = rateLimit({
  windowMs: number,         // time window in ms
  max: number,              // max requests per window per key
  standardHeaders: boolean, // send RateLimit-* headers (RFC 6585), includes Retry-After
  legacyHeaders: boolean,   // send X-RateLimit-* headers (disable = false)
  message: object | string, // response body when limit exceeded
  keyGenerator: (req) => string, // how to identify client
});
```
