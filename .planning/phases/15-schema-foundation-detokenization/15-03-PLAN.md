<objective>
Detokenize `server/lib/openrouter.ts`. Add an optional `companyName?: string` parameter to both exported factory functions (`getOpenRouterClient` and `listOpenRouterModels`) and replace the literal `|| "Skleanings"` fallback with parameter-injected resolution `process.env.OPENROUTER_APP_TITLE || companyName || ""`. Purpose: SERV-01 requires the OpenRouter HTTP `X-Title` header to reflect the actual tenant identity rather than a hardcoded brand string. Without this change, every OpenRouter request from this server identifies the application as "Skleanings" regardless of which tenant is running.

Output: Detokenized openrouter factory + 3 updated caller sites + DI type plumbing.
</objective>
