<!-- Patterns from 39-RESEARCH.md — use these directly for all method edits -->

SELECT (no existing WHERE):
```typescript
// Before
return await db.select().from(categories).orderBy(categories.order);

// After
return await db.select()
  .from(categories)
  .where(eq(categories.tenantId, this.tenantId))
  .orderBy(categories.order);
```

SELECT (existing WHERE condition):
```typescript
// Before
const baseConditions = [eq(services.isArchived, false)];
return await db.select().from(services).where(and(...baseConditions));

// After
const baseConditions = [
  eq(services.tenantId, this.tenantId),  // prepend
  eq(services.isArchived, false),
];
return await db.select().from(services).where(and(...baseConditions));
```

INSERT:
```typescript
// Before
const [newCat] = await db.insert(categories).values(category).returning();

// After
const [newCat] = await db.insert(categories)
  .values({ ...category, tenantId: this.tenantId })
  .returning();
```

UPDATE / DELETE:
```typescript
// Before
.where(eq(categories.id, id))

// After
.where(and(eq(categories.tenantId, this.tenantId), eq(categories.id, id)))
```

Singleton SELECT + INSERT (getCompanySettings pattern):
```typescript
const [settings] = await db.select().from(companySettings)
  .where(eq(companySettings.tenantId, this.tenantId));
if (settings) return settings;
const [newSettings] = await db.insert(companySettings)
  .values({ tenantId: this.tenantId })
  .returning();
return newSettings;
```

Transaction with multiple inserts (createBooking):
- Every `tx.insert(bookings).values(...)` → spread `tenantId: this.tenantId`
- Every `tx.insert(bookingItems).values(...)` → spread `tenantId: this.tenantId`
- Every `tx.update(bookings).where(eq(bookings.id, ...))` → wrap with `and(eq(bookings.tenantId, this.tenantId), eq(bookings.id, ...))`
