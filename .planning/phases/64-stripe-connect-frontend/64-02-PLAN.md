<objective>
Surface every tenant's Stripe Connect status to the super-admin panel (SC-07). Add a "Connect" column to the SuperAdmin Tenants table showing connection state and capability flags so platform operators can audit Connect adoption without DB inspection.

Purpose: Without this, super-admin has no way to see which tenants have onboarded with Stripe Connect — a critical operational signal for v19.0 and the upcoming v20.0 payment routing work.

Output: Extended `GET /api/super-admin/tenants` aggregate response, updated `TenantListItem` type, and new column in the Tenants table.
</objective>
