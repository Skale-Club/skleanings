Pattern for parameter threading:
```typescript
// BEFORE
import { storage } from "../storage";
export async function doThing(): Promise<void> {
  const data = await storage.getCompanySettings();
}

// AFTER — remove import, add parameter
import type { IStorage } from "../storage";
export async function doThing(storage: IStorage): Promise<void> {
  const data = await storage.getCompanySettings();
}
```

Exception — auth.ts has Express middleware functions (requireAdmin, requireClient, getAuthMe):
These functions have fixed Express signatures (req, res, next). They cannot accept a storage parameter
in the middleware position. Instead, read storage from res.locals inside the function body:
```typescript
// auth.ts middleware pattern
export async function requireAdmin(req: Request, res: Response, next: NextFunction) {
  const storage = res.locals.storage!;
  const user = await getAuthenticatedUser(req, storage);
  // ...
}
```
