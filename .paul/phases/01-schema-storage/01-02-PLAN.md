  <name>Task 1: Add staffMemberId to bookings table in shared/schema.ts</name>
  <files>shared/schema.ts</files>
  <action>
    In the `bookings` table definition (around line 100), add `staffMemberId` as the last field before the closing `}`  ‚Äî after the `ghlSyncStatus` field:

    ```
    staffMemberId: integer("staff_member_id").references(() => staffMembers.id),
    ```

    No `.notNull()` ‚Äî the column must be nullable so existing bookings (which have no staff) are not affected.
    No `onDelete: "cascade"` ‚Äî if a staff member is deleted, we want to keep the booking record (set to null instead). Use `onDelete: "set null"` on the FK:

    ```
    staffMemberId: integer("staff_member_id").references(() => staffMembers.id, { onDelete: "set null" }),
    ```

    Avoid: Adding `.notNull()` ‚Äî would break all existing bookings.
    Avoid: Using `onDelete: "cascade"` ‚Äî deleting a staff member must not delete their booking history.
