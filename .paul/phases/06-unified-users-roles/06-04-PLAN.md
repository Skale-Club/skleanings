<objective>
## Goal
When a user is created or updated with role='staff', automatically create a linked staffMembers record (userId FK set). When a staff user is deleted, clean up the linked staffMembers record first to avoid FK constraint violation.

## Purpose
Without this bridge, creating a user with role='staff' only creates a users row — no staffMembers record means they won't appear in the booking flow, availability engine, or calendar management. This plan closes that gap.

## Output
- `server/storage.ts` — new `getStaffMemberByUserId` method (IStorage + DatabaseStorage)
- `server/routes/user-routes.ts` — POST creates staffMember on role=staff; PATCH creates staffMember if role→staff; DELETE cleans up staffMember before deleting user
</objective>

```gherkin
Given a POST /api/users request with role='staff', firstName, lastName, email
When the request is processed
Then a users row is created AND a staffMembers row is created with userId pointing to the new user

```

## AC-2: Role change to staff creates staffMembers record
```gherkin
Given a PATCH /api/users/:id request changing role to 'staff'
When the user previously had no linked staffMembers record
Then a staffMembers record is created with userId = that user's id
And if a staffMembers record already exists for this userId, no duplicate is created
```

## AC-3: Staff user deletion cleans up staffMembers first
```gherkin
Given a DELETE /api/users/:id request for a user with role='staff'
When the user has a linked staffMembers record (userId FK)
Then the staffMembers record is deleted first
And then the user record is deleted (no FK constraint violation)
```

    import { Router } from "express";
    import { storage } from "../storage";
    import { requireAdmin } from "../lib/auth";

    const router = Router();

    router.get("/", requireAdmin, async (req, res) => {
        const users = await storage.getUsers();
        res.json(users);
    });

    router.post("/", requireAdmin, async (req, res) => {
        const newUser = await storage.createUser(req.body);

        if (newUser.role === 'staff') {
            const created = await storage.createStaffMember({
                firstName: newUser.firstName || '',
                lastName: newUser.lastName || '',
                email: newUser.email || undefined,
                phone: newUser.phone || undefined,
                profileImageUrl: newUser.profileImageUrl || undefined,
                isActive: true,
                order: 0,
            });
            await storage.updateStaffMember(created.id, { userId: newUser.id });
        }

        res.status(201).json(newUser);
    });

    router.patch("/:id", requireAdmin, async (req, res) => {
        const id = req.params.id;
        const updates = req.body;

        // Prevent demoting the last admin
        if (updates.role && updates.role !== 'admin') {
            const user = await storage.getUser(id);
            if (user && user.role === 'admin') {
                const users = await storage.getUsers();
                const adminCount = users.filter(u => u.role === 'admin').length;
                if (adminCount <= 1) {
                    return res.status(400).json({ message: "Cannot remove the last administrator." });
                }
            }
        }

        const updated = await storage.updateUser(id, updates);

        // Bridge: if role changed to staff and no staffMember linked yet, create one
        if (updates.role === 'staff') {
            const existing = await storage.getStaffMemberByUserId(updated.id);
            if (!existing) {
                const created = await storage.createStaffMember({
                    firstName: updated.firstName || '',
                    lastName: updated.lastName || '',
                    email: updated.email || undefined,
                    phone: updated.phone || undefined,
                    profileImageUrl: updated.profileImageUrl || undefined,
                    isActive: true,
                    order: 0,
                });
                await storage.updateStaffMember(created.id, { userId: updated.id });
            }
        }

        res.json(updated);
    });

    router.delete("/:id", requireAdmin, async (req, res) => {
        const id = req.params.id;
        const user = await storage.getUser(id);

        if (user && user.role === 'admin') {
            const users = await storage.getUsers();
            const adminCount = users.filter(u => u.role === 'admin').length;
            if (adminCount <= 1) {
                return res.status(400).json({ message: "Cannot delete the last administrator." });

        const linkedStaff = await storage.getStaffMemberByUserId(id);
        if (linkedStaff) {
            await storage.deleteStaffMember(linkedStaff.id);
        }

        await storage.deleteUser(id);
        res.status(204).send();
    });
