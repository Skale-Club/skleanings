<objective>
## Goal
Replace the tabbed UnifiedUsersSection with a single flat users table; update UserDialog to use a role picker (admin/user/staff) instead of an isAdmin toggle; fix role badges in UsersSection; update user-routes to use role logic.

## Purpose
Phase 1 shipped `role` on the users table. This plan wires that role field into the UI: admins see everyone in one list with role badges, and the Add/Edit dialog lets them assign roles directly.

## Output
- `UnifiedUsersSection.tsx` — no tabs, renders flat users list
- `UsersSection.tsx` — role badges from `user.role` (admin/user/staff)
- `UserDialog.tsx` — role Select instead of isAdmin Switch; admin-only 'admin' option
- `user-routes.ts` — PATCH/DELETE guards use `role === 'admin'` instead of `isAdmin`
</objective>
