  <action>
    Create StaffManageDialog component.

    Props:
    ```typescript
    interface StaffManageDialogProps {
      staffId: number;
      staffName: string;
      open: boolean;
      onOpenChange: (open: boolean) => void;
    }
    ```

    Structure:
    - Dialog with DialogContent (sm:max-w-lg)
    - DialogHeader: "{staffName} — Settings"
    - Tabs (default: "services"): TabsList with "Services" and "Availability"

    **Services tab (TabsContent value="services"):**
    - useQuery: GET /api/services (key: ['/api/services']) — returns Service[]
    - useQuery: GET /api/staff/:id/services (key: ['/api/staff', staffId, 'services']) — returns Service[]
    - Local state: Set<number> of selected serviceIds, initialized from assigned query
    - useEffect to sync selectedIds when assignedServices loads
    - Show each service as a row: Checkbox + service name (use service.name field)
    - Save button: PUT /api/staff/:id/services with { serviceIds: [...selectedIds] }
    - invalidate ['/api/staff', staffId, 'services'] on success, toast "Services updated"

    **Availability tab (TabsContent value="availability"):**
    - useQuery: GET /api/staff/:id/availability (key: ['/api/staff', staffId, 'availability'])
    - Local state: array of 7 items `{ dayOfWeek: number; isAvailable: boolean; startTime: string; endTime: string }`
    - useEffect to build 7-item state from query data. For each dayOfWeek 0–6:
      - Find matching record from query, else default: { isAvailable: dayOfWeek >= 1 && dayOfWeek <= 5, startTime: '09:00', endTime: '17:00' }
    - Render 7 rows in a grid (grid-cols-[auto_1fr_auto_auto] gap-3 items-center):
      - Day name label (Sun, Mon, Tue, Wed, Thu, Fri, Sat)
      - Switch for isAvailable
      - Time Input for startTime (disabled when !isAvailable, type="time")
      - Time Input for endTime (disabled when !isAvailable, type="time")
    - Save button: PUT /api/staff/:id/availability with the 7-item array
    - invalidate ['/api/staff', staffId, 'availability'] on success, toast "Availability updated"

    Imports needed:
    - Dialog, DialogContent, DialogHeader, DialogTitle from ui/dialog
    - Tabs, TabsContent, TabsList, TabsTrigger from ui/tabs
    - Checkbox from ui/checkbox
    - Switch from ui/switch
    - Input from ui/input
    - Button from ui/button
    - Label from ui/label
    - Loader2 from lucide-react
    - useQuery, useMutation from @tanstack/react-query
    - apiRequest, queryClient from @/lib/queryClient
    - useToast from @/hooks/use-toast
    - type Service from @shared/schema

    Day names array: ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat']
