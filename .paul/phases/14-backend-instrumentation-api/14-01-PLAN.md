# Phase 14 Plan 01: Notification Logger Helper + Twilio + Telegram Instrumentation

## Goal

Every Twilio SMS and Telegram message send writes a row to `notification_logs`. A shared helper
(`logNotification`) wraps the storage call so it never throws — notifications must never fail
because of logging. Twilio logs one row per phone number; Telegram logs one row per chatId.

## Acceptance Criteria

| ID | Criterion | Done When |
|----|-----------|-----------|
| AC-1 | `logNotification` never throws | try/catch wraps `storage.createNotificationLog`; error goes to `console.error` only |
| AC-2 | Twilio `sendNewChatNotification` logs per phone | One row per phone with channel=sms, trigger=new_chat, status=sent/failed, providerMessageId=Twilio SID, conversationId |
| AC-3 | Twilio `sendCalendarDisconnectNotification` logs per phone | One row per phone with trigger=calendar_disconnect, no conversationId/bookingId |
| AC-4 | Twilio `sendBookingNotification` logs per phone | One row per phone with trigger=new_booking, optional bookingId when caller provides it |
| AC-5 | Telegram `sendMessageToAll` logs per chatId | One row per chatId when logContext provided; status reflects per-chatId success/failure |
| AC-6 | Test message creates no rows | `sendTelegramTestMessage` calls `sendMessageToAll` without logContext → no DB writes |
| AC-7 | TypeScript compiles | No new type errors; all new params are optional (no call-site breakage) |

## Tasks

### Task 1 — Create `server/lib/notification-logger.ts`

**File to create:** `server/lib/notification-logger.ts`

```ts
import { storage } from "../storage";
import type { InsertNotificationLog } from "@shared/schema";

const MAX_PREVIEW_LENGTH = 5000;

export async function logNotification(entry: InsertNotificationLog): Promise<void> {
  try {
    const safeEntry: InsertNotificationLog = {
      ...entry,
      preview: entry.preview.slice(0, MAX_PREVIEW_LENGTH),
    };
    await storage.createNotificationLog(safeEntry);
  } catch (err) {
    console.error("[notification-logger] Failed to write log entry:", err);
  }
}
```

**Verification:** File exists at correct path. `logNotification` is exported.
